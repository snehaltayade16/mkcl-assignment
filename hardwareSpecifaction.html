Hardware Specifications

  1. Minimum Lab setup :1 Server + 10 Clients
  2. Xtenda Setup is not allowed

  1. For Desktop:- CPU - Intel Core i3-10100(10th Gen) @ 3.60GHz / AMD Ryzen 5 3600
  2. For Laptop:- CPU - Intel Core i3-1005G1 @ 1.20GHz / AMD Ryzen 5 5500U
  3. RAM:- 16 GB
  4. Nvme/SSD: 1 TB (Only for Server)
  5. Integrated Graphics
  6. LAN Cable for server (Wi-Fi not recommended)
  7. 3 USB Ports
  8. Windows 11(64-bit) Pro/Ent Edition
  9. Antivirus
  10. Quantity required percenter (minimum): 1 Desktop/Laptop
  11. Minimum 100 Mbps Broadband Internetconnection

The above-mentioned server can support up to 20 Clients. If the number of Clients is more, then the server CPU and RAM should be increased proportionately

  1. Desktop:- CPU - Intel Core i3-10100(10th Gen) @ 3.60GHz / AMD Ryzen 5 3600
  2. Laptop:- CPU - Intel Core i3-1005G1 @ 1.20GHz / AMD Ryzen 5 5500U
  3. RAM:- 8 GB
  4. Nvme/SSD: 256 GB
  5. Lan Cable / Wi-Fi Network
  6. Audio ports
  7. Keyboard and Mouse
  8. Headphone with Mic
  9. One Web Camera / integrated Camera
  10. 3 USB ports
  11. Minimum 15inch Monitor or above
  12. Windows 11 Pro/Enterprise (64-bit)
  13. Antivirus
  14. Quantity required per center (minimum): 10 Desktops/Laptops*
  15. Laptops are preferable because:
    1. Consume less power
    2. Require less space
    3. Easy to maintain
    4. Minimum 3-4 hours of power backup available in case of power failure
    5. Inbuilt Web camera
    6. Easy to move
    7. WiFi connectivity

  1. Printer - minimum 600 dpi resolution
  2. Scanner - minimum 600 dpi resolution
  3. Bio Metric Device (minimum 1)
  4. Power Backup for all Computers
  5. UPS or
  6. Inverter with Battery backup(Compulsory)
  7. DG Set (Optional)
  8. CCTV (Compulsory)
  9. Projector or LCD Display (Optional)
  10. Speakers (Optional)

  1. LAN 1 Gbps Network Switch (Server and all Laptops should be connected to this Network Switch) or Dual Band WiFi Access Point – 300 Mbps, 802.11n
  2. Electrical and LAN wiring should run perpendicular to each other. If they are parallel to each other, then a distance of 2 feet should be maintained between them to avoid interference

  1. Minimum 100 Mbps Broadband Connection (wired / wireless) (Server and all Laptops should be able to connect to the Internet by using Internet Connection Sharing or Proxy Server)
